# Employee Data Analysis

## Overview
This project analyzes employee data to extract insights related to salaries,
departments, cities, and experience levels using Python and Pandas.

The goal of this project is to demonstrate practical data analysis skills
commonly used in real-world business scenarios.

## Dataset
The dataset contains employee information including:
- Name
- City
- Department
- Salary
- Years of Experience

## Tools Used
- Python
- Pandas
- Jupyter Notebook

## Key Analysis
- Average salary by department
- Average salary by city
- Identification of high-salary employees
- Feature engineering (salary level classification)
